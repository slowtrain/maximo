# AGENTS.md

이 저장소에서 작업하는 Codex, Claude 등 모든 코드·문서 에이전트가 따르는 공통 규칙입니다.

## 저장소 목적

IBM Maximo Application Suite(MAS) 설치 및 구성 절차를 문서화합니다.  
코드 구현이 아닌 **운영 문서** 저장소입니다.

## 문서 작업 규칙

- 사용자에게는 항상 한국어 존댓말로 답변합니다.
- 문서는 실제 절차와 일치해야 합니다. 미검증 내용은 `> ⚠️ 미검증` 블록으로 표시합니다.
- 계획·메모는 `docs/notes/` 아래에 두고 주요 문서에 섞지 않습니다.
- 기존 문서를 수정할 때는 변경 이유를 커밋 메시지에 남깁니다.
- 명령어 예시는 반드시 코드 블록(```` ```bash ````) 으로 작성합니다.
- 환경별로 달라지는 값은 `<placeholder>` 형식으로 표시합니다.

## 문서 구조 원칙

| 경로 | 용도 |
|------|------|
| `docs/installation/` | 설치 절차 문서 |
| `docs/configuration/` | 구성 및 설정 문서 |
| `docs/operations/` | 운영·유지보수 문서 |
| `docs/notes/` | 작업 메모·미완성 초안 |

## 금지 사항

- 실제 비밀번호·API 키·Entitlement Key를 문서에 직접 기재하지 않습니다.
- 검증되지 않은 명령어를 단정적으로 기술하지 않습니다.
- 문서 파일에 바이너리·이미지를 직접 커밋하지 않습니다.
